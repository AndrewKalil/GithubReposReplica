# HelloBuild Dev test

Let's get started...
1. clone the project
```
git clone https://github.com/AndrewKalil/GithubReposReplica.git
```
2. Enter Project directory
```
cd ./GithubReposReplica
```

3. run proxy server
```
cd ./proxy-server
node ./serve.js
```

4. Open up another terminal and run frontend app
```
cd ./githubrepos
npm start
```
